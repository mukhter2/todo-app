# To-do app

This project uses JavaScript,MongoDB and requires Node.js and npm to be installed on your system.
### Prerequisites

You should install `Node`, `git`

### Installing

`git clone url`

`cd url`

### Backend Running

go to backend directory using `cd todo-app`
then run `npm install`

For development:
`cp .env.dev .env`

`nodemon index.js`
